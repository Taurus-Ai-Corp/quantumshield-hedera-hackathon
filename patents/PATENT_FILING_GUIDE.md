## üöÄ Filing Process

### Step 1: Complete Patent Information

For each patent, fill in:
- Inventor name(s)
- Filing date (upon submission)
- Entity status (Micro-Entity recommended)

### Step 2: Prepare for USPTO EFS-Web Filing

**Prerequisites:**
1. Create USPTO account: https://www.uspto.gov/patents/apply/my-account
2. Obtain Customer Number (unique USPTO ID)
3. Set up EFS-Web access: https://efs-web-signon.uspto.gov/

### Step 3: File Provisional Patents

**Filing Method: USPTO EFS-Web**

1. **Login to EFS-Web:**
   - Go to: https://efs-web-signon.uspto.gov/
   - Login with USPTO account

2. **Start New Application:**
   - Select "New Application"
   - Choose "Provisional Application for Patent"
   - Select "Micro-Entity" status (if eligible)

3. **Upload Documents:**
   - Upload patent specification (markdown converted to PDF)
   - Add cover sheet (auto-generated or manual)
   - Include any drawings/diagrams

4. **Pay Filing Fee:**
   - Micro-Entity: $70 per application
   - Small Entity: $140 per application
   - Regular Entity: $280 per application
   - **Total (5 patents, micro-entity): $350**

5. **Submit and Confirm:**
   - Review all information
   - Submit application
   - Save filing receipt
   - Note application number

### Step 4: Track Applications

- Use USPTO Private PAIR to track status
- Monitor application numbers
- Set reminders for 12-month deadline (to file non-provisional)
